## 📖 Sobre o Repositório

Este espaço foi criado para organizar e versionar todo o código fonte produzido durante as aulas da disciplina. Aqui você encontrará desde os primeiros scripts para fixação de lógica até projetos mais elaborados construídos ao longo do semestre.

## 🛠️ Tecnologias e Ferramentas

Os projetos deste repositório foram desenvolvidos utilizando as seguintes ferramentas:

* **[PHP](https://www.php.net/):** Linguagem de programação principal utilizada no backend.
* **[Visual Studio Code](https://code.visualstudio.com/):** Editor de código (IDE) escolhido para o desenvolvimento.
* **[XAMPP](https://www.apachefriends.org/pt_br/index.html) / Servidor Local:** (Opcional: cite aqui se você usa Apache/MySQL localmente para rodar o PHP).

## 📂 Estrutura de Pastas

Os arquivos estão organizados de forma cronológica e por tópicos para facilitar a navegação:

* 📁 `exercicios-de-fixacao/`: Introdução ao PHP, variáveis e estruturas de repetição.
* 📁 `exercicios-avaliativos/`: Manipulação de métodos GET e POST, estruturação de CRUDS.
* 📁 `trabalho-pratico/`: Arquivos referentes aos trabalhos avaliativos e finais.

> **Nota:** Cada pasta de projeto maior contém seu próprio arquivo de anotações ou documentação específica, se necessário.

## 🚀 Como executar os projetos localmente

Se você quiser clonar e rodar esses códigos na sua máquina, siga os passos abaixo:

1. Certifique-se de ter um servidor local instalado (como XAMPP, WAMP ou Laragon) rodando o **Apache**.
